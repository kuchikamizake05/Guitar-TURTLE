# Guitar Turtle Drawing

Proyek ini adalah implementasi grafis sederhana menggunakan modul **Turtle** di Python untuk menggambar sebuah gitar akustik secara prosedural.

## Fitur
- Menggunakan fungsi geometri dasar (Lingkaran, Elips, Poligon).
- Menggambar bagian gitar secara detail: Body, Sound Hole, Bridge, Neck, Headstock, dan Senar.
- Fitur ekspor otomatis ke format `.eps` saat layar diklik.

## Cara Menjalankan
1. Pastikan Anda sudah menginstal Python.
2. Jalankan skrip:
   ```bash
   python gitar.py
   ```
3. Klik pada jendela aplikasi untuk menyimpan hasil gambar ke file `gitar.eps` dan menutup program.

## Prasyarat
- Python 3.x
- Modul `turtle` (sudah termasuk dalam standar library Python).
